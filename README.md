# NeuroPlan

**AI-powered diagnostic and adaptive study planning engine.**

---

## Overview

NeuroPlan is an AI-based system that combines diagnostic assessment with structured study scheduling.

Instead of generating generic study advice, NeuroPlan:

- Evaluates student understanding  
- Identifies weaknesses  
- Recommends a study mode  
- Produces a structured 7-day plan  

---

## Problem

Students often:

- Misjudge their strengths and weaknesses  
- Spend excessive time planning  
- Follow generic AI advice that is not actionable  

Most AI tools provide text suggestions but do not convert performance data into structured execution plans.

**NeuroPlan focuses on turning diagnosis into action.**

---

## Solution

NeuroPlan integrates three stages:

1. Constraint-aware setup  
2. Diagnostic evaluation  
3. Structured plan generation  

### Inputs

- Exam date  
- Project deadlines  
- Daily available hours  
- Subject difficulty  
- Diagnostic results (generated by the program and also used as an input) 

### Outputs

- Weakness insights  
- Mode recommendation (Autopilot or Coach)  
- A 7-day structured study plan  

---

## How It Works

### 1. Setup

Users provide:

- Exam date  
- Deadlines  
- Daily study hours  
- Subject and difficulty  

These define the planning constraints.

---

### 2. Diagnostic Quiz

The system generates seven subject-specific questions:

- Mix of multiple-choice and short-answer  
- Includes a "No idea" option  
- Strict JSON output format  
- Validated using Zod schemas  
- Retry and fallback handling  

This ensures structured and reliable AI output.

---

### 3. Insight and Mode Recommendation

Based on performance:

- Conceptual weaknesses are identified  
- A study mode is recommended  

**Autopilot mode** provides stronger structure.  
**Coach mode** provides lighter guidance.

---

### 4. Plan Generation

NeuroPlan generates a 7-day plan including:

- Daily focus areas  
- Time allocation within constraints  
- Concrete tasks and deliverables  
- Built-in review components  

The plan is designed to be executable, not descriptive.

---

## AI Implementation

NeuroPlan includes:

- Structured prompt engineering  
- Enforced JSON output  
- Zod schema validation  
- JSON extraction and sanitation  
- Timeout handling  
- Retry logic  
- Performance-based branching  
- Constraint-aware plan generation  

**Model used:** MiniMax API  

All AI responses are validated before rendering.
